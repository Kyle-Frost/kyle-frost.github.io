The Descent

A 3-week Game Jam based on a given theme - "A horror game where you explore dungeons full of trolls and loot sweet gear."

This game jam features randomly generated dungeons with varying designed rooms, created so each run will feel unique in it's design. Equipped with a bow and elemental arrows, you can take down enemies to spawn randomised loot with power tiers - making you more powerful as you push through each floor of the dungeon. At the end of each dungeon floor is a boss with unique attacks to provide an extra challenge.

DirectX FPS

A first person shooter developed in C++ using DirectX.

This basic FPS game includes collision detection, billboard enemies and shooting mechanics made from scratch. Cubes are loaded in individually and texture mapped to create a level, with their positions, and thus the level design, decided using a text file. Collision detection was created by comparing vertex data to create a basic FPS game from scratch.

Duality

A 1-week Game Jam for the 2022 Global Game Jam with the theme "Duality".

Using Photon for Unity, this game jam features a functional lobby system, allowing users to create and name rooms to host private games. Other players can browse open rooms and join from a list in the menu. The host of the game also has the options to change gamemode and map for that specific game.

Our FPS game features multiple different weapons, including hitscan and projectile. The weapon system is modular, using Scriptable Objects to manage weapon data, created 2 different gamemodes - a typical deathmatch and a "looter-shooter mode", as well as 2 different maps to play on. Weapons models although basic, were created myself using Unity cubes and emmisive materials to appeal to a futuristic theme.

Hidden in Plain Sight

A C++ game created on the theme "Hacking", using the custom ASGE engine.

Hidden in Plain Sight is a party game created with heavy inspiration from Adam Spragg's console game of the same name. The game is local multiplayer, with one player playing as the hacker, and the other as police. The role of the hacker is to hack all 5 terminals without being detected, whilst the job of the police is to find and eliminate the hacker without harming too many civilians.

The game scene has many wandering AI to allow the hacker to blend in, and make it harder for the police to track down the hacker. But the terminals will light up when hacked to help the police narrow down their search. Utilising smoke grenades and masking their human movement is important for the hacker to evade, whilst marking innocent civilians will allow the police to close in on the hacker.

Kojima Wars

A cohort Unity FPS/RTS game based around the classic game "Herzog Zwei".

Kojima wars is a FPS and RTS hybrid, with mech-based combat including various weapons and abilities to provide players with different playstyle options. The game also features RTS elements, including base capture and defending. These bases earn currency in which you can spend on base defences, or AI units to deploy. A mix of combat skill and intelligence is important to succeed.

Movement is based around the classic Herzog Zwei game, with the mech able to transform from mech to VTOL, allowing the player to fly around the map and attack bases from different angles. The rules of the game are classic domination - capture bases to earn points on a ticker, and the team who reaches the score limit first, wins!

Our team, Foundation Games, was in charge of the weapon system and abilities. For this, I developed a modualar weapon system, allowing for the creation of multiple unique weapons with many adjustable stats. Combined with a procedural recoil system, this made gun play feel dynamic and engaging. Alongside this, I developed a semi-modular skills/ability system, allowing for the creation of skills within 4 classes - Assault, Movement, Defence and Recon - each of which had a distinctive playstyle. Examples of abilities include an Airstrike, Smoke grenade, Ring of Fire, Decoy Clone, Dash and many more!
